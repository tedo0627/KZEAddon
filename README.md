# KZEAddon
kzeのプレイをサポートするmodです  
forgeとfabric両方対応  
version 1.19.4

## 機能
 - 同じチームのプレイヤーを透明にする ※1
 - 明るさを最大にする

※1 キー設定で指定したキーを押すことでオンオフ可能

## 設定方法
 - キーの設定はマイクラのデフォルトのキー設定から変更可能
 - /kzeaddonで設定を開く
