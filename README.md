# KZEAddon
kzeのプレイをサポートするmodです  
version 1.14.4

## 機能
・同じチームのプレイヤーの透明化(人間の時人間が透明になったり、ゾンビの時ゾンビが透明になったりします)  
・弾の残り数を表示  
・リロード時間の表示  
・過去のキルログを遡れる

## 使い方
oキーを押すと設定画面が開かれます  
nキーを押すと同じチームのプレイヤーが透明化します  
kキーを押すとキルログを表示します  
キーを変えたい場合はマイクラのキー設定から変更してください
